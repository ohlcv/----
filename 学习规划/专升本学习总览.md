## 🧠 跨学科学习方法

### 1. 核心类比体系

| 计算机科学概念 | 神经科学对应概念 | 核心功能 |
|--------------|----------------|---------|
| 晶体管/逻辑门 | 神经元 | 基本计算单元，输入累加→阈值触发→放电 |
| 连接权重/网络 | 突触 | 决定信号传递强度，可动态调整（学习的物理基础） |
| CPU/操作系统内核 | 前额叶皮层(PFC) | 决策、计划、抑制冲动、工作记忆 |
| 中断控制器/警报系统 | 杏仁核 | 威胁检测、情绪中断、强制抢占 |
| 内存(RAM) | 海马体 | 临时存储、编码、短期记忆 |
| 硬盘/SSD | 大脑皮层 | 长期记忆、技能固化 |
| 高速总线/屏蔽层 | 髓鞘 | 提升信号传递速度、抗干扰 |
| 后台守护进程/空闲线程 | 默认模式网络(DMN) | 胡思乱想、复盘、内耗 |
| 系统信号/IPC | 神经递质(多巴胺/血清素等) | 调节注意力、情绪、动机 |

### 2. 关键学习原理

- **赫布定律**：一起放电的神经元连在一起（Cells that fire together, wire together）
- **突触可塑性**：突触强度可动态调整，是学习和记忆的基础
- **睡眠的关键作用**：通过睡眠将短期记忆巩固为长期记忆
- **注意力机制**：注意力 = PFC独占调度 + 屏蔽中断
- **分布式存储**：记忆分布在整个神经网络中，而非特定区域

### 3. 大脑与计算机的本质区别

- **模拟+数字混合**：神经元信号是数字脉冲，突触传递是化学模拟
- **低精度、高容错**：单个神经元不可靠，但网络整体稳健
- **能耗极低**：人脑约20W，远超同等复杂度AI
- **存算一体**：记忆储存在突触连接中，计算即激活模式传播
- **自我构造**：学习过程与结构变化紧密耦合

### 2. 核心学习流程（7步闭环）

1. **考纲结构化拆解**：建立知识地图，标注考点权重
2. **习题分层驱动学习**：从易到难，先独立思考再问答
3. **苏格拉底式深度追问**：暴露知识盲区，而非仅"听懂答案"
4. **即时输出验证**：通过变式题和费曼技巧防止假性理解
5. **间隔复习**：遵循艾宾浩斯遗忘曲线
6. **知识网络编织**：将零散知识点连成体系
7. **模拟考试+元认知复盘**：分析错题类型和学习策略漏洞

## 💡 防坑指南

1. **避免AI依赖**：先独立思考10分钟，明确问题后再问AI
2. **验证知识准确性**：涉及专业/考试的核心知识点，对照教材、考纲验证
3. **警惕"假性理解"**：通过输出验证确保真正掌握，而非"AI讲得明白"
4. **防止知识碎片化**：定期梳理知识框架，建立连接
5. **保持学习主动性**：AI只是工具，学习的核心是主动建构知识
