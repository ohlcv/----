# 考试向学习笔记 · 元模板（应试工程化 · 注释版）

> 核心理念：**模板固定，模块功能固定，内容由考点资料和考纲填充**。
> 目标：**最小认知负荷 → 考试可用 → 支撑复习与理解**

---

## 一、定位与权重（全局坐标）

* 教材章名：`【第*章 ********】`
* 教材节名：`【第*节 ********】`
* 知识点名：`【知识点 * ********】`
* 考点星级：★ / ★★ / ★★★
* 常见题型：单选 / 多选 / 简答 / 论述
* 能力层级：识记 → 理解 → 应用

> 🎯 功能：确定考点优先级和复习深度
> **注释**：填入考纲中明确提及的章节、题型和星级，明确学习与刷题重点。

【类比】

* 计算机：模块优先级 / 资源调度器
* 神经科学：注意力分配 / 工作记忆优先级
